Create a Function Inside a Struct

Easy

Problem Description

Create a program to find the average marks of a student.

Step 1:

  • Create a struct named Student with an array int marks[3] and a function average_marks() that returns a double value.
  • Calculate the average of the elements of marks[] array inside the average_marks() function.

Step 2:

  • In main() get user input for marks[] array using a for loop.
  • Print the average marks by calling the average_marks() function.
